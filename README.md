# Kiwifan1.github.io

This is the official website of Kiwifan1 (Joshua), built with Angular.

## Development

This project uses Angular 20. To run locally:

```bash
npm install
npm start
```

Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.

## Build

To build the project for production:

```bash
npm run build
```

The build artifacts will be stored in the `dist/browser/` directory.

## Deployment

The site is automatically deployed to GitHub Pages when changes are pushed to the `main` branch via GitHub Actions.

## Structure

- **Home** - Main landing page
- **About** - About me page
- **Links** - Social media and other links
- **Resume** - Resume viewer
- **Tools** - Collection of tools organized by category
  - **Minecraft** - Minecraft-related tools and utilities (more coming soon)
